Atlas details#

Introduction#

The atlases are available in a standardised form, via the BrainGlobe Atlas API (or any software that uses it). However, each atlas varies slightly in the versions that are available (e.g. resolutions).

To use the atlases with the API directly, the unique key (e.g. example_mouse_100um must be used).
